Bienvenue sur Roboc-game__inoino.2.0 Multiplayer!


1. Lancer le serveur en premier, puis selectionner la map. 
=> serveur.py

2. Lancer 2 à 4 clients, puis se connecter au serveur.
=> client.py


Remarques : 

- Possibilite de jouer à partir de machines distantes.
=> modifier ligne 38 de client.py : entrer l'adresse ip de l'ordinateur sur lequel
le serveur est lance (à la place de 'localhost').

- Tests unitaire disponibles (à lancer à partir de console)


Bon jeu.
